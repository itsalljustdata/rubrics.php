Help Instructions

  1. Log into Blackboard and access the unit (Instructor access is needed).

  2. Scroll down to the Unit Management menu, click and expand Packages and Utilities and click on Export/Archive Unit.

  3. Click the Archive button.

  4. Make sure the Include Grade Centre History (increases file size and processing time) option is checked.

    IMPORTANT: Make sure the Copy only links to unit defualt directory files (first option, NOT like the screenshot below)

    Then click the Submit button.

  5. When the Archive file has processed you will be able to download it on the Export/Archive Unit screen (see step 2). Once the archive file appears on the list, click it to download.

  6. Use this file on the main page to create the rubric reports.
